# Lab 10 Solutions — Reindex, Session Management, Memory & Advanced Features

This directory contains reference solutions for Lab 10, the capstone lab that covers reindex (semantic code understanding), Memory MCP for persistent decisions/context, session handoff techniques, continuous learning, and advanced Copilot CLI features.

## Files

| File | Lab Section | Purpose |
|------|-------------|---------|
| `memory-entities.json` | 10.2 | Sample Memory MCP entities — decisions and context (not code facts — reindex handles those) |
| `memory-relations.json` | 10.2–10.3 | Sample Memory MCP relations linking entities |
| `sample-handoff.md` | 10.5 | Example session handoff document generated by `/handoff` |
| `sample-checkpoint-log.txt` | 10.5 | Example checkpoint log entry from `/checkpoint create` |

## Key Concepts

- **Reindex** automatically understands your code — structure, types, relationships, patterns — no configuration needed
- **Memory MCP** is complementary — use it for decisions, conventions, and knowledge NOT in the code itself
- **Continuous learning v1** extracts patterns at session end; **v2** observes every tool call via hooks
- **Handoff** (`/handoff`) creates a comprehensive session-end document; **Checkpoint** (`/checkpoint`) is a lightweight progress marker
- **LSP**, **semantic search**, **sub-agents**, and **fleet mode** are advanced features that amplify each other
