# Example Skill

Invoke this skill to verify the plugin is wired up. It prints the plugin
manifest name and version and exits successfully.

## Usage
Call this skill from any agent in the plugin. It expects no inputs.

## Notes
- Keep skill bodies short — a skill is a capability prompt, not a workflow.
- Skills compose: one plugin can bundle many skills for the same domain.
